Project Details - Figma

C218 CA - StyleShop

In this module, I was tasked to design a user friendly mobile application called StyleShop, the application allows users to carry out basic navigation, the icons are meant to be intuitive.

Designing this StyleShop Figma project, I thought about furniture as a lifestyle, considering that furniture can depict someone's style and preferences too. Thinking about furniture, I am reminded of the colour peach, which represents sweetness, goodwill, and joy.
